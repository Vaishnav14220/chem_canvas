📍 STEP 2: OPEN DOCUMENT DESIGNER
┌──────────────────────────────────────────────────────────────────────────────┐
│ 1. Make sure app is running: http://localhost:1754                          │
│ 2. Click "Show Study Tools" button (right panel)                            │
│ 3. Navigate to "Designer" tab                                               │
└──────────────────────────────────────────────────────────────────────────────┘

📍 STEP 3: CONFIGURE API KEY
┌──────────────────────────────────────────────────────────────────────────────┐
│ 1. Look for ORANGE Settings button (⚙️) in top toolbar                      │
│ 2. Click the Settings button                                                │
│ 3. Paste your API key in the input field                                    │
│ 4. Click "Save API Key"                                                     │
│ 5. Settings button turns from orange to normal = SUCCESS! ✅                │
