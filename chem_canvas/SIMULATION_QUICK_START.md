# Quick Start Guide: Dynamic Simulation Generator

## 🎯 What is it?

The Dynamic Simulation Generator automatically converts your educational documents into interactive, model-based simulations. Upload a PDF about physics, chemistry, biology, or finance, and watch as AI creates interactive calculators and simulations!

## 🚀 Quick Start (3 Steps)

### Step 1: Upload Your Document
1. Click **"Document Understanding Workspace"** in the main app
2. Click the upload button and select a PDF or text file
3. Wait for the document to be processed

### Step 2: Generate Simulations
1. Once your document is processed, click **"Generate Simulations"** button
2. The AI will analyze your document (takes 10-30 seconds)
3. You'll see a list of suggested interactive simulations

### Step 3: Create & Interact
1. Click on any simulation suggestion
2. Wait for the simulation to configure (5-10 seconds)
3. Use sliders, knobs, and switches to adjust parameters
4. Watch results update in real-time!

## 🎨 Available Control Types

### Sliders
- Best for: Continuous values with defined ranges
- Example: Voltage (0-24V), Temperature (-100 to 500°C)
- Features: Visual track, min/max labels, current value display

### Knobs
- Best for: Dynamic adjustment with visual feedback
- Example: Frequency, Phase angle, Concentration
- Features: Circular progress indicator, numeric display

### Switches/Toggles
- Best for: On/Off states, Enable/Disable features
- Example: AC/DC mode, Include air resistance, Use ideal conditions
- Features: Smooth animation, clear visual state

### Number Fields
- Best for: Precise numeric entry
- Example: Exact measurements, Specific quantities
- Features: Keyboard input, unit labels, min/max validation

### Dropdowns
- Best for: Categorical choices
- Example: Element selection, Calculation mode, Units
- Features: List of options, clear labels

## 📊 Output Formats

### Decimal
- Standard format: `12.345`
- Best for: General calculations

### Scientific Notation
- Format: `1.23e-4`
- Best for: Very large/small numbers

### Percentage
- Format: `45.67%`
- Best for: Rates, efficiencies, yields

### Currency
- Format: `$1,234.56`
- Best for: Financial calculations

## 🔧 Tips & Tricks

### For Best Results:
1. **Clear Documents**: Use documents with well-defined formulas and explanations
2. **Structured Content**: Documents with headings and organized topics work best
3. **Formula Clarity**: LaTeX or clearly written mathematical equations
4. **Definitions**: Include term definitions for better context

### Troubleshooting:
- **No Simulations Generated?** 
  - Ensure your document has mathematical formulas
  - Check that the document is readable (not scanned images)
  
- **Calculation Errors?**
  - Check input constraints (min/max values)
  - Ensure units are consistent
  
- **Slow Generation?**
  - Large documents take longer to process
  - Complex formulas require more analysis time
